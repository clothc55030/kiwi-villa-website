📝 功能說明

房號跳轉邏輯：
  • www.kiwi-villa.com/201 → 跳轉到房間頁面並定位到 201 房間
  • www.kiwi-villa.com/302 → 跳轉到房間頁面並定位到 301 房間（302與301共用卡片）
  • www.kiwi-villa.com/306 → 跳轉到房間頁面並定位到 305 房間（306、307與305共用卡片）
  • 支援所有房號：201, 203, 205, 301, 302, 303, 305, 306, 307
⚠️ 注意事項：
  • 此功能僅在 Cloudflare Pages 生產環境中有效
  • 本地測試時重定向不會生效，請發佈後測試
  • 房間卡片會有3秒的高亮效果以引起注意
  • 重定向使用 _redirects 文件配置

🛠 技術實現

實現方式：
  1. Cloudflare Pages 重定向：在 _redirects 文件中配置房號重定向規則
  2. 301永久重定向：/205 → /rooms?goto=205
  3. 房間定位：在 main.js 中處理查詢參數並定位房間
  4. 高亮效果：使用 CSS 動畫效果高亮顯示目標房間3秒
🚀 Cloudflare Pages 優勢：
  • 邊緣重定向：在 Cloudflare 全球邊緣節點進行重定向，超快響應
  • SEO 友好：301重定向告訴搜索引擎這是永久重定向
  • 更可靠：不依賴瀏覽器 JavaScript 支持
  • 全球 CDN：利用 Cloudflare 全球網絡加速
