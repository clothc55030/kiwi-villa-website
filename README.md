## 使用方法

### 本地開發
1. 下載所有文件到本地目錄
2. 使用任何HTTP服務器運行（如Live Server）
3. 在瀏覽器中開啟 `index.html`

### 部署
1. 上傳所有文件到網站服務器
2. 確保所有路徑正確指向
3. 替換佔位圖片為實際圖片
